## 📌 Project Overview

This project is a pixel-perfect implementation of a Figma design using HTML, CSS, and JavaScript.

The landing page includes:

* Responsive Navigation Bar
* Mobile Sidebar Menu
* Hero Section
* Compliance Information Section
* Interactive Buttons
* Hover Effects
* Mobile, Tablet, and Desktop Support
